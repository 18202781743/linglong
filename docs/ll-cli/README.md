#ll-cli user guide
ll-cli是一个包管理器前端.

```bash
ll-cli --help
```

```plain
Usage: ll-cli [options] subcommand [sub-option]

Options:
  -h, --help        Displays this help.
  --default-config  default config json filepath
  --nodbus          execute cmd directly, not via dbus

Arguments:
  subcommand        run
                    ps
                    exec
                    kill
                    download
                    install
                    uninstall
                    update
                    query
                    list
```
